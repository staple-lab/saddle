Everything you need

Professional component authoring that generates AI-friendly, production-ready output.

T

Token-First Design

Edit via tokens only. Saddle enforces consistency across your design system and auto-suggests tokenization for raw values.

AI

AI-Readable Components

design.md frontmatter gives AI explicit metadata: when to use components, constraints, and best practices.

MCP

Claude Code Integration

Bidirectional MCP server lets Claude Code read, modify, and create components directly through Saddle.

HMR

Live Dev Server Preview

Embed your running localhost app inside Saddle. Edit tokens, file saves to disk, HMR updates your real app instantly.

DOM

Element Inspector

Chrome DevTools-style element tree showing your component's full DOM hierarchy with computed styles per element.

npm

npm Package Export

One-click export with conditional exports, generated CSS Modules, global tokens.css, and DTCG format.

Get started in 5 minutes

1

Load your project

Point Saddle at any React component library. It auto-detects component directories and parses design.md frontmatter.

2

Connect dev server

Saddle auto-detects your running localhost server and embeds it. Or use isolated preview mode.

3

Edit with tokens

Figma-style property inspector with token pickers. Changes save to disk and your app updates via HMR.

4

Connect Claude Code

One-click MCP setup. Claude reads your component schemas, updates tokens, and creates variants.
